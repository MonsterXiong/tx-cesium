## ⚙️ 内网 npm 配置

### 方案 1：npm 私服

```bash
npm config set registry http://your-npm-registry
npm install
```

### 方案 2：离线安装

```bash
# 有网环境
npm pack cesium

# 内网环境
npm install cesium-1.137.0.tgz
npm install
```

### 方案 3：复制 node_modules

```bash
# 从有网环境复制
tar -czf node_modules.tar.gz node_modules/
# 解压到内网环境
```

## 🐛 常见问题

### Q: 地图显示黑屏？

**检查**：
```bash
# 确认 Cesium 资源存在
ls dist/cesium/Assets/

# 确认资源可访问
curl http://localhost/cesium/Cesium.js
```

### Q: Workers 加载失败？

**解决**：
```nginx
# Nginx 添加 MIME 类型
location /cesium/Workers/ {
    types { application/javascript js; }
}
```

### Q: 安装依赖失败？

**原因**：内网无法访问 npm 官方源

**解决**：配置内网私服或使用离线包
