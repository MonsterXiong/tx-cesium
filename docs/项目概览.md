## 🏗️ 架构设计

### 1. 路由架构
```
配置式路由系统
├── routes/config.tsx      - 路由配置和元信息
├── routes/index.tsx       - 路由主入口
└── 特性
    ├── 懒加载
    ├── 代码分割
    ├── 路由元信息
    └── 自动生成菜单
```

### 2. 状态管理架构
```
Zustand 轻量级状态管理
├── useThemeStore     - 主题状态（亮/暗模式、主题色）
├── useUserStore      - 用户状态（登录、用户信息）
└── useAppStore       - 应用状态（侧边栏、面包屑）
```

### 3. 主题系统架构
```
CSS Variables + Antd Theme
├── styles/variables.css   - CSS 变量定义
├── useTheme Hook         - 主题管理 Hook
└── ConfigProvider        - Antd 主题配置
```

### 4. Cesium 集成架构
```
Cesium 封装层
├── CesiumViewer 组件      - Viewer 封装
├── cesium/config.ts      - 配置管理
├── cesium/utils/         - 工具函数
└── useCesium Hook        - Cesium Hook
```
