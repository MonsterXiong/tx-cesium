# 代码规范 - 快速参考卡

## 📝 注释

```typescript
/**
 * 函数说明
 * @param name 参数说明
 * @returns 返回值说明
 */
const func = (name: string): void => {}

// Console 带前缀
console.log('[DEBUG_USER] 说明:', data)
```

## 🎨 CSS

```tsx
// ✅ 使用 UnoCSS 工具类 + CSS 变量
<div className="flex-center p-4 text-primary bg-bg-container">

// ✅ 使用预定义快捷方式
<div className="flex-between card-base transition-base">
```

**快捷方式**：
- `flex-center` - flex 居中
- `flex-between` - 两端对齐
- `card-base` - 卡片样式
- `transition-base` - 过渡动画

## 🔤 命名

### 文件命名

| 文件类型 | 规范 | 示例 |
|---------|------|------|
| React 组件 | PascalCase | `UserCard.tsx`, `BasicLayout.tsx` |
| 页面文件夹 | kebab-case | `cesium-view/`, `not-found/` |
| 页面主文件 | `index.tsx` | `index.tsx` |
| Hooks | camelCase | `useTheme.ts`, `useCesium.ts` |
| 工具函数 | camelCase | `storage.ts`, `helpers.ts` |
| 样式文件 | kebab-case | `cesium-viewer.css`, `global.css` |
| 配置文件 | camelCase/kebab-case | `config.ts`, `vite.config.ts` |
| 类型定义 | kebab-case.d.ts | `api.d.ts`, `global.d.ts` |

### 代码命名

| 类型 | 规范 | 示例 |
|------|------|------|
| 常量 | UPPER_SNAKE_CASE | `MAX_COUNT` |
| 函数 | 动词开头 | `handleClick`, `fetchData` |
| 变量 | camelCase | `userName`, `isLoading` |
