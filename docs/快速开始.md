## 🚀 启动步骤

### 1. 安装依赖（已完成）
依赖已经安装完成，无需再次执行。

### 2. 配置环境变量（可选）

如果需要使用 Cesium 地图的完整功能，请配置 Token：

1. 访问 [Cesium Ion](https://ion.cesium.com/) 注册账号
2. 创建 Access Token
3. 创建 `.env.development` 文件（已有模板）：

\`\`\`env
VITE_APP_TITLE=React Template
VITE_APP_BASE_API=/api
VITE_CESIUM_BASE_URL=/cesium
VITE_CESIUM_ACCESS_TOKEN=你的_token_这里
\`\`\`

### 3. 启动开发服务器

\`\`\`bash
npm run dev
\`\`\`

项目将在 http://localhost:3000 启动

### 4. 构建生产版本

\`\`\`bash
npm run build
\`\`\`

### 5. 预览生产版本

\`\`\`bash
npm run preview
\`\`\`

## 🎯 核心功能使用

### 1. 添加新页面

\`\`\`typescript
// 1. 在 src/pages/ 创建页面组件
// 2. 在 src/routes/config.tsx 添加路由配置
{
  path: '/new-page',
  meta: {
    title: '新页面',
    icon: <YourIcon />,
  },
}

// 3. 在 routeComponents 添加组件映射
const NewPage = lazy(() => import('@/pages/new-page'))
export const routeComponents = {
  '/new-page': NewPage,
}
\`\`\`

### 2. 使用状态管理

\`\`\`typescript
import { useThemeStore, useUserStore, useAppStore } from '@/stores'

// 主题管理
const { mode, toggleMode } = useThemeStore()

// 用户管理
const { userInfo, login, logout } = useUserStore()

// 应用状态
const { sidebarCollapsed, toggleSidebar } = useAppStore()
\`\`\`

### 3. 使用主题

\`\`\`typescript
import { useTheme } from '@/hooks'

const { mode, antdThemeConfig, toggleMode } = useTheme()
\`\`\`

### 4. 使用 Cesium

\`\`\`typescript
import CesiumViewer from '@/cesium/CesiumViewer'

<CesiumViewer
  onViewerReady={(viewer) => {
    console.log('Cesium Viewer 已准备就绪', viewer)
  }}
/>
\`\`\`

## 📝 开发建议

1. **Console 调试标识**: 项目中的 console 已统一添加 `[DEBUG_*]` 前缀，方便在控制台搜索过滤
2. **类型定义**: 在 `src/types/` 中定义类型，保持类型安全
3. **API 接口**: 在 `src/services/api/` 中定义接口，统一管理
4. **工具函数**: 在 `src/utils/` 中添加通用工具函数
5. **组件复用**: 在 `src/components/` 中创建可复用组件
