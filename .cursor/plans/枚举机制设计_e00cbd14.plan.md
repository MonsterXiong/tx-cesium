# 枚举机制设计方案

## 背景/目标

- 定义枚举值和中文标签，支持颜色、options、value 映射
- 使用方式简化：`getLabel('orderStatus', value)`，无需单独引入枚举
- 找不到枚举时返回空，不报错
- **命名约定：枚举名称统一使用小驼峰（camelCase）**

## 命名约定（小驼峰）

- 枚举常量名：小驼峰，如 `orderStatus`、`userRole`
- 调用时传入的字符串：小驼峰，如 `getLabel('orderStatus', value)`、`getOptions('orderStatus')`
- 注册表 key：小驼峰，如 `enumRegistry.orderStatus`

## 目录结构

```
src/enums/
├── index.ts           # 统一导出 + 枚举注册表（脚本生成）
├── helper.ts          # 枚举工具函数
└── modules/
    ├── status.ts      # 状态相关枚举
    ├── user.ts        # 用户相关枚举
    └── ...
```

## 枚举定义格式（小驼峰）

```typescript
// src/enums/modules/status.ts
export const orderStatus = {
  PENDING: { value: 'pending', label: '待处理', color: 'orange' },
  PROCESSING: { value: 'processing', label: '处理中', color: 'blue' },
  COMPLETED: { value: 'completed', label: '已完成', color: 'green' },
  CANCELLED: { value: 'cancelled', label: '已取消', color: 'red' },
} as const
```

## 工具函数

通过**枚举名称字符串（小驼峰）**访问，找不到时静默返回空：

```typescript
getLabel('orderStatus', 'pending')   // => '待处理'
getLabel('notExist', 'xxx')         // => ''

getColor('orderStatus', 'pending')  // => 'orange'
getItem('orderStatus', 'pending')   // => { value, label, color } 或 undefined
getOptions('orderStatus')           // => [{ value, label }, ...]
getValues('orderStatus')           // => ['pending', 'processing', ...]
getEnum('orderStatus')             // => 枚举对象
```

## 实现原理

脚本生成的 `index.ts` 中，注册表 key 为小驼峰：

```typescript
import { orderStatus } from './modules/status'
import { userRole, userStatus } from './modules/user'

export const enumRegistry: Record<string, any> = {
  orderStatus,
  userRole,
  userStatus,
}

export { orderStatus } from './modules/status'
export { userRole, userStatus } from './modules/user'
export * from './helper'
```

## 使用示例

```tsx
import { getOptions, getLabel, getColor } from '@/enums'

<Select options={getOptions('orderStatus')} />
<span>{getLabel('orderStatus', record.status)}</span>
<Tag color={getColor('orderStatus', record.status)}>
  {getLabel('orderStatus', record.status)}
</Tag>

// 直接访问枚举值
import { orderStatus } from '@/enums'
if (status === orderStatus.PENDING.value) { ... }
```

## 脚本

- `scripts/generate-enums.ts`：扫描 `modules/`，解析 `export const xxx = { ... } as const`，生成 import、enumRegistry（key 小驼峰）、export，写入 `index.ts`
- 约定：导出名即为注册表 key，需使用小驼峰命名
